# Kimberly Druist

Single-page personal site for Kimberly Druist.

## Local Dev

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy

Deploys automatically via Vercel on push to `main`.

## Launch Checklist

- About copy
- Positioning
- Three case studies
- Essay
- Photo
- Meta description
- OG image
- Favicon
