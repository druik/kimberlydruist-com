# Kimberly Druist

Single-page personal site for Kimberly Druist.

## Local Dev

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy

Deploys automatically via Vercel on push to `main`.

## Launch Checklist

- TK: About copy
- TK: positioning
- TK: three case studies
- TK: essay
- TK: photo
- TK: meta description
- TK: OG image
- TK: favicon
